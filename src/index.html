CSS Classes

Add Class

Function for adding class to element:

Copy to clipboard 
                                
$('div').addClass('some-class');
Remove Class

Function for remove class from element:

Copy to clipboard 
                                
$('div').removeClass('some-class');
Toggle Class

Function for toggle class on element(work with events):

Copy to clipboard 
                                
$('.g-btn').on('click', function () {               
    $('div').eq(2).toggleClass('active');
});

Actions

Copy to clipboard 
                                
$('.some-class').html(`Hi, ${person.title}`);

Copy to clipboard

Copy to clipboard

Handlers

ON method

Allows you to hang various events on an element:

Copy to clipboard 
                                
$('.t-daynight').on('click', function() {
    $('html').toggleClass('m-dark-scheme');
});
OFF method

Disable event on element:

Copy to clipboard 
                                
$('.t-daynight').off('click', function() {
    $('html').toggleClass('m-dark-scheme');
});
Click event

Added click event to element:

Copy to clipboard 
                                
$('.g-header').click(function() {
    console.log($(this).index());
});

Effects

FadeIn

Method gradually changes the opacity, for selected elements, from hidden to visible (fading effect):

Copy to clipboard 
                                
$('.some-div').fadeIn(2000);
FadeOut

Method gradually changes the opacity, for selected elements, from visible to hidden (fading effect):

Copy to clipboard 
                                
$('.some-div').off('click', function() {
    $('.more').fadeOut(2000);
});
Click event

Added click event to element:

Copy to clipboard 
                                
$('.g-header').click(function() {
    console.log($(this).index());
});

    </div>
</div>

Modal Extended

This method for extended modal window:

Copy to clipboard 
                                
// JS Init Extended Initialization                                   
$('[data-target="modal-3"]').click(() => $('[data-target="modal-3"]').createModal({
    text: {
        title: 'Modal Title',
        content: 'Some Body'
    },
    btns: {
        count: 2, // Numbers of btns
        settings: [
            [
                'close window', // Text button
                ['g-button', 'btn-close'], // Add classes to button
                true // Indicates closing button
            ],
            [
                'Save changes', // Text button
                ['g-button', 'btn-send'], // Add classes to button
                false, // Indicates closing button
                () => {
                    alert('Function')
                }
            ] 
        ]
    }
}));
